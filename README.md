# SUL Styles

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'sul_styles'
```

And then execute:

    $ bundle install

Import the SUL Styles in your `app/assets/stylesheets/application.scss`.

    @import 'sul-styles'
